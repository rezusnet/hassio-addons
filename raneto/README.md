# Raneto

[![GitHub Release][releases-shield]][releases]
[![License][license-shield]](LICENSE.md)

Raneto is an open source knowledge base platform powered by static Markdown files. This add-on wraps the LinuxServer.io Raneto image for Home Assistant.

## Features

- Clean, searchable interface for documentation
- Powered by static Markdown files (git-friendly)
- Self-hosted wiki for smart home documentation
- Optional HTTP port for direct access
- Home Assistant Ingress support

## Installation

1. Add this repository to Home Assistant: https://github.com/rezusnet/hassio-addons
2. Install the Raneto add-on
3. Configure data location and timezone
4. Start the add-on
