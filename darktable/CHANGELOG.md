# Changelog

All notable changes to this project will be documented in this file.

## [25.4.1] - 2026-04-30

### Added

- Initial Darktable add-on release
- Support for amd64 and aarch64 architectures
- HTTPS web interface on port 3001
- Data storage configuration with configurable location
- Timezone support
- Local and network disk mounting support
- CIFS (SMB) network share support

### Features

- Darktable 25.4.1 based on linuxserver/darktable
- Home Assistant ingress support
- S6 overlay integration
- Health check monitoring
