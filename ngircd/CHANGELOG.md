# Changelog

All notable changes to the NGircd Home Assistant add-on will be documented in this file.

## [27.1] - 2025-01-27

### Added
- Initial Home Assistant add-on release
- Full IRC protocol support (RFC 2812/2813)
- User authentication and channel management
- TLS/SSL encryption on port 6668
- Configurable server parameters
- Custom MOTD (Message of the Day) support
- Connection limits and throttling
- Comprehensive documentation

### Features
- Multi-architecture support (amd64, aarch64)
- S6 init system integration
- Home Assistant add-on standards compliance
- Persistent configuration storage
- TCP health checks
- Log level configuration
