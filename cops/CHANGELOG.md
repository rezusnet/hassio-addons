## 4.3.1 (27-04-2026)

- Initial release of COPS add-on
- Based on LinuxServer.io Docker image (4.3.1)
- Support for amd64 and aarch64 architectures
- OPDS feed support for e-readers and mobile devices
- Web interface for book browsing and management
- SMB/CIFS network share mounting
- Local disk mounting support
- Timezone and UID/GID configuration
- HA Ingress sidebar integration
