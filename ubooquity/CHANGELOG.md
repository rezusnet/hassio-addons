# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [3.1.0] - 2026-04-30

### Added

- Initial release of Ubooquity add-on for Home Assistant
- Support for amd64 and aarch64 architectures
- Web ingress support for library interface (port 2202)
- Admin interface access (port 2203)
- Configurable storage location
- Memory limit configuration via MAXMEM option
- Support for timezone configuration
- Support for local and network disk mounting
- Security profile (AppArmor)

### Features

- Free, lightweight home server for comics and ebooks
- Access library from anywhere with tablet, e-reader, phone, or computer
- Automatic file scanning at startup
- Home Assistant integration with data persistence in /share/ubooquity
