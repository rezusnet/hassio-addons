# Home Assistant Add-on: Ubooquity

![Project Badge](https://img.shields.io/badge/linuxserver-ubooquity-informational)
![Support Badge](https://img.shields.io/badge/supports-amd64%2Caarch64-blue)
![Version](https://img.shields.io/badge/version-3.1.0-green)

[Ubooquity](https://vaemendis.net/ubooquity/) is a free, lightweight, and easy-to-use home server for your comics and ebooks. Use it to access your files from anywhere, with a tablet, an e-reader, a phone or a computer.

![Banner](banner.svg)

## Installation

Follow these steps to install the add-on:

1. Navigate to Home Assistant Settings → Add-ons → Add-on Store
2. Search for "Ubooquity" and select it
3. Click the "Install" button
4. Configure the add-on options if needed
5. Start the add-on

## Getting Started

1. Start the Ubooquity add-on
2. Access the admin interface at `http://<your-home-assistant>:2203/ubooquity/admin`
3. Set a password to secure your instance
4. Access the library at `http://<your-home-assistant>:2202/ubooquity/`
5. Add your comics and ebooks to the configured library paths
6. Enjoy browsing your collection from anywhere!

## Key Features

- 📚 Manage your comics and ebooks collection
- 🌐 Web-based access from any device
- 📱 Tablet, e-reader, phone, or computer compatible
- ⚙️ Automatic file scanning at startup
- 💾 Lightweight and efficient
- 🔐 Admin interface with password protection
- 🌍 Access your library from anywhere
- 🏠 Full Home Assistant integration
