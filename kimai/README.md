## About

Kimai is an open source time tracking application designed for small teams and freelancers. It provides comprehensive time tracking, project and customer management, with features including invoice generation, expense tracking, and detailed reporting.

## Key Features

- ⏱️ **Time Tracking**: Track time spent on tasks and projects
- 📊 **Project Management**: Organize work into projects and customers
- 👥 **Multi-User Support**: Multiple users can track their own time
- 📄 **Invoicing**: Generate invoices from tracked time
- 💰 **Expense Tracking**: Track expenses alongside time entries
- 📈 **Detailed Reports**: View comprehensive time and expense reports
- 🔐 **Self-Hosted**: Keep your data private on your own server

## Installation

1. Add the add-on repository (if not already added)
2. Install the Kimai add-on from the add-on store
3. Configure the add-on (see DOCS.md for details)
4. Start the add-on
5. Access Kimai via the Home Assistant sidebar or directly at the configured ingress port

## Quick Access

- **Web Interface**: Available through Home Assistant Ingress or at `http://<your-host>:80`
- **Default Credentials**: Check the add-on logs for the default admin account information
