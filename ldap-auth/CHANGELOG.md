# Changelog

## 3.4.5 (30-04-2026)

- Initial release of LDAP Auth add-on
- LDAP and Active Directory authentication support
- Cookie-based session management with Fernet encryption
- Nginx reverse proxy integration
- SSL/TLS support for the auth daemon
- Custom Fernet key support for multi-instance deployments
- Support for amd64 and aarch64 architectures
