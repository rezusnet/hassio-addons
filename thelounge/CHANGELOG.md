## [1.0.0] - 2026-04-30

### Added

- Initial release of TheLounge add-on for Home Assistant
- Web-based IRC client with persistent connections
- Support for multiple IRC networks and users
- Theme customization and user preferences
- Message history and logging
- Mobile-friendly interface
- Ingress support for seamless HA integration
- Configuration options for PUID, PGID, timezone, and data location
- Health checks for service monitoring

### Features

- Full IRC protocol support
- Channel and user auto-completion
- Customizable appearance and themes
- User account management
- Network management with easy setup
- Browser-based access from any device
- Secure TLS/SSL connection support for IRC networks
