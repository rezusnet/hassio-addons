# Changelog

All notable changes to the Your-Spotify Home Assistant add-on will be documented in this file.

## [1.8.0] - 2025-01-27

### Added
- Initial Home Assistant add-on release
- Spotify API integration with secure OAuth authentication
- Playlist analysis and visualization
- Streaming statistics tracking
- Web-based UI with responsive design
- Home Assistant ingress support for direct UI access
- Configurable data location with persistence
- Support for external MongoDB backend
- JWT-based session management
- Comprehensive documentation

### Changed
- Based on LinuxServer.io Your-Spotify Docker image
- Optimized for Home Assistant environment

### Features
- Multi-architecture support (amd64, aarch64)
- S6 init system integration
- Home Assistant add-on standards compliance
