# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.6.2] - 2026-04-30

### Added

- Initial pyLoad-ng Home Assistant add-on release
- Based on LinuxServer.io pyload-ng Docker image
- Web interface accessible via Home Assistant Ingress
- SMB/CIFS network share mounting support
- Local disk mounting support
- Timezone configuration
- Persistent configuration and logs storage
- Support for x86-64 (amd64) and ARM64 (aarch64) architectures
- Comprehensive documentation and configuration options

### Features

- Lightweight download manager
- Multi-threaded downloads
- Plugin support
- Web-based control interface
- Cross-platform compatible
