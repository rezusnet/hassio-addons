## About

[Your Spotify](https://github.com/linuxserver/docker-your) is a powerful application for managing and organizing media content.

This add-on is based on the [linuxserver/docker-your](https://github.com/linuxserver/docker-your) Docker image.

**Key features:**

- Easy installation and configuration
- Multi-architecture support (amd64, aarch64)
- Home Assistant ingress support
- Secure AppArmor profile
- Full documentation

## Installation

1. Add this repository to your Home Assistant instance
2. Install the **Your Spotify** add-on
3. Configure the add-on options
4. Start the add-on
5. Access via the HA sidebar or directly

For full configuration details, see the **Documentation** tab.
