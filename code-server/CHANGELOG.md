## 4.117.0 (30-04-2026)

- Initial release of Code Server add-on based on linuxserver/code-server
- Support for password and hashed password authentication
- Configurable data location for workspace storage
- PGID/PUID user mapping for file permissions
- Timezone support
- Docker mods support for additional extensions
- Custom environment variables support
- HA ingress sidebar integration
- Platform diagnostics at startup: Pi model detection, memory and disk space logging
