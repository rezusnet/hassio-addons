## About

[Telegram](https://github.com/linuxserver/docker-telegram) is a powerful application for managing and organizing media content.

This add-on is based on the [linuxserver/docker-telegram](https://github.com/linuxserver/docker-telegram) Docker image.

**Key features:**

- Easy installation and configuration
- Multi-architecture support (amd64, aarch64)
- Home Assistant ingress support
- Secure AppArmor profile
- Full documentation

## Installation

1. Add this repository to your Home Assistant instance
2. Install the **Telegram** add-on
3. Configure the add-on options
4. Start the add-on
5. Access via the HA sidebar or directly

For full configuration details, see the **Documentation** tab.
