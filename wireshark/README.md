## About

Wireshark is the world's foremost network protocol analyzer. It lets you capture and interactively browse the traffic running on a computer network. Wireshark is the de facto standard for protocol analysis used by network engineers and IT professionals worldwide.

## Key Features

- 🔍 **Deep Packet Inspection**: Analyze network traffic at the packet level
- 🌐 **Protocol Support**: Decode hundreds of network protocols
- 📊 **Real-time Capture**: Capture and analyze traffic as it happens
- 🎯 **Powerful Filtering**: Filter and search captured packets
- 📈 **Traffic Analysis**: Generate statistics and reports
- 🔐 **SSL/TLS Support**: Decrypt encrypted traffic with keys
- 🎨 **Graphical Interface**: Browser-based GUI via KASMVNC

## Installation

1. Add the add-on repository (if not already added)
2. Install the Wireshark add-on from the add-on store
3. Configure the add-on (see DOCS.md for details)
4. Start the add-on
5. Access Wireshark via the Home Assistant sidebar or directly at the configured port

## Quick Access

- **Web Interface**: Available through Home Assistant Ingress or at `http://<your-host>:3000`
- **Desktop Environment**: Complete Linux desktop with Wireshark application
