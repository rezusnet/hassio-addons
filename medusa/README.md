# Medusa Add-on

[![Release][release-badge]][release-link]
[![License][license-badge]][license-link]

Automatic Video Library Manager for TV Shows.

## About

[Medusa][medusa] is an automatic Video Library Manager for TV Shows. It watches for new episodes of your favorite shows, and when they are posted it does its magic.

## Features

- Automatic episode downloads using NZB or Torrent
- Planned series status management
- Episode status tracking
- Post-processing support
- Anime support
- Media center integrations
- Extensive configuration options

## Installation

The installation of this add-on is straightforward and easy to do.

1. Add the repository to the Home Assistant add-on store: `https://github.com/rezusnet/hassio-addons`
2. Install the "Medusa" add-on
3. Configure the add-on with your preferences
4. Start the add-on
5. Open the Web UI
